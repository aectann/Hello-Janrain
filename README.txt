To get it up and running you need Maven 2.2+.

To run:
 0. set up a Jainrain account and obtain an apiKey and application domain name
 1. set your Jainrain apiKey in web.xml
 2. set your domain name in index.jsp
 3. cd to project root
 4. run 'mvn jetty:run'
 5. open 'http://localhost:8080' in your browser
